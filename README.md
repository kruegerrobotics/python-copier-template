# Python Project Template

## Purpose

This is a collection of files that represents a template for any new Python project - regardless if for quick prototyping or larger projects. It has a series of lints and static code checks with pre-made pipelines that should ensure code consistency internal or across collaboration. It also helps to create required boilerplate for any good repo with pre-commit, pipelines etc.

It also can be applied with care to existing projects.

## Quickstart

### New project from an existing (empty)folder

If you are already familiar with this template, just write the following to create a new project.

```bash
mkdir my-project
cd my-project
```
